# 快速开始

本指南将帮助您快速上手单词记忆卡片 API。我们将介绍如何查询单词、处理响应数据，以及如何在不同编程语言中使用 API。{{ className: 'lead' }}

<Note>
  本 API 完全公开，无需认证即可使用。每个 IP 每天有 500 次免费请求额度。

## 基础查询 {{ id: 'basic-query' }}

最简单的方式是使用浏览器或 cURL 直接访问 API：

<CodeGroup>

```bash {{ title: 'cURL' }}
curl "https://api.keykey.cc/api/public/memory-card?word=hello"
```

```bash {{ title: '浏览器' }}
# 直接在浏览器中访问
https://api.keykey.cc/api/public/memory-card?word=hello
```

</CodeGroup>

## 使用示例 {{ id: 'code-examples' }}

以下是在不同编程语言中调用 API 的示例：

<CodeGroup>

```js {{ title: 'JavaScript' }}
async function getMemoryCard(word, lang = 'en') {
  const url = `https://api.keykey.cc/api/public/memory-card?word=${encodeURIComponent(word)}&lang=${lang}`
  
  const response = await fetch(url)
  const data = await response.json()
  
  if (data.success) {
    console.log('单词:', data.data.word)
    console.log('音标:', data.data.phonetic)
    console.log('例句:', data.data.examples)
    console.log('剩余请求次数:', data.rateLimit.remaining)
  } else {
    console.error('错误:', data.error)
  }
  
  return data
}

// 使用示例
const card = await getMemoryCard('hello')
```

```python {{ title: 'Python' }}
import requests

def get_memory_card(word: str, lang: str = 'en'):
    url = 'https://api.keykey.cc/api/public/memory-card'
    params = {'word': word, 'lang': lang}
    
    response = requests.get(url, params=params)
    data = response.json()
    
    if data['success']:
        print(f"单词: {data['data']['word']}")
        print(f"音标: {data['data']['phonetic']}")
        print(f"例句: {data['data']['examples']}")
        print(f"剩余请求次数: {data['rateLimit']['remaining']}")
    else:
        print(f"错误: {data['error']}")
    
    return data

# 使用示例
card = get_memory_card('hello')
```

```php {{ title: 'PHP' }}
<?php
function getMemoryCard($word, $lang = 'en') {
    $url = 'https://api.keykey.cc/api/public/memory-card?' . 
           http_build_query(['word' => $word, 'lang' => $lang]);
    
    $response = file_get_contents($url);
    $data = json_decode($response, true);
    
    if ($data['success']) {
        echo "单词: " . $data['data']['word'] . "\n";
        echo "音标: " . $data['data']['phonetic'] . "\n";
        echo "剩余请求次数: " . $data['rateLimit']['remaining'] . "\n";
    } else {
        echo "错误: " . $data['error'] . "\n";
    }
    
    return $data;
}

// 使用示例
$card = getMemoryCard('hello');
?>
```

</CodeGroup>

## 错误处理 {{ id: 'error-handling' }}

API 可能返回以下错误：

<Properties>
  <Property name="NOT_FOUND (404)">
    单词尚未生成记忆卡片，可能是生僻词或尚未收录。
  </Property>
  <Property name="RATE_LIMIT_EXCEEDED (429)">
    超过每日请求限额（500次），将在第二天 UTC 0:00 重置。
  </Property>
  <Property name="INVALID_PARAMETER (400)">
    参数错误，请检查 word 参数是否为有效字符串。
  </Property>
  <Property name="INTERNAL_ERROR (500)">
    服务器内部错误，请稍后重试。
  </Property>
</Properties>
